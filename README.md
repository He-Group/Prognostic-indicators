# Prognostic-indicators
## Folder [Analysis]
- *00_data cleaning WCH* imports the managed datasets and finalizes data preprocessing of the WCH cohort
- *01_prognostic model_WCH OS* describes the development and evaluation of prognostic prediction model for OS in the WCH cohort
- *02_prognostic model_WCH DFS* describes the development and evaluation of prognostic prediction model for DFS in the WCH cohort
- *03_prognostic model validation_UKB OS*  describes the evaluation of prognostic prediction model for OS in the UKB cohort
- *04_cox*  explores the association between plasma proteins and survival outcomes using Cox regression in the WCH cohort
- *05_pQTLs selection*  selects the pQTLs of identified plasma proteins
- *06_MR*  MR analysis
- *07_MR_sensibility*  MR Steiger test, Bayesian colocalization analysis and MRlap
- *08_single-cell analysis*  Single-cell analysis of PD-L1
- *09_spatial_transcriptomics* Spatial transcriptomic analysis of PD-L1
